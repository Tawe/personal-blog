## 🔧 Local Development

Want to run this locally? Here's how:

bash

```bash

git clone https://github.com/Tawe/personal-blog.git

# Install dependencies
npm install

# Start the development server
npm run dev
```

Then visit `http://localhost:3000` to see the blog in action.
